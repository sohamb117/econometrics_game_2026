Two-way fixed effects event-study DID model on time-series data to estimate the long-term causal impact of a financial literacy education policy requirement on adult credit outcomes, leveraging high-dimensional unit and time fixed effects to isolate policy-induced variation.
Leverages staggered treatment cohorts and estimated dynamic treatment effects via event-time indicators, evaluating post-policy adjustment and temporal heterogeneity in treatment effect.
Validated identification assumptions using parallel-trend diagnostics. Finalized all studies into a 12-page econometric research paper.
